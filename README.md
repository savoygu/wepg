## 项目搭建

1. 创建 git 项目，然后 clone 到本地

```bash
git clone
```

2. 初始化 package.json 文件

```bash
npm init
```

3. 创建 `.editorconfig` 文件

```bash
# 安装 editorconfig-cli
npm install -g editorconfig-cli

# 初始化配置
ec init
```

4. 创建 `.eslintrc.json` 文件

```
# 安装 eslint
npm install eslint --save-dev

# 初始化配置
npx eslint --init
```
